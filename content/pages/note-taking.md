## Updates

### 2024-03

Have been used several days of **GitHub Project** and found it helps me to remove the project management from note taking app, previously I keep currently working items with Obsidian and use a DataView to extract into a table, it makes Obsidian to do something not very good at(some people may disagree), for me I must stop to find a app can do all those things at once. Same reason I won't use Orgmode in Emacs again.

### 2024-05

Using **Apple Notes** for work, instead of Foam plugin in VSCode. I usually need to open multiple VSCode windows for projects, Foam make it hard to find, this annoys me, everthing else are OK. Apple Notes annoys me more, but I still want to give it a try, and found some pros:

- Folder with icons
- Smart folder: I can filter notes with uncompleted tasks quickly
- Tags
- Table
- Photo
- Gallary view
- Attachements browser
- Shortcuts integration

**Obsidian** on my iPhone was getting stuck when opening and loading, moving notes to **Apple Notes** as well, since I don't have too much personal notes, I can live with the Apple Notes, it not attracting compares to any of the alternatives but it is reliable. Beside personal notes, I keep all my professional notes into a GitHub repo with consist configurations, docker-compose files, etc etc.

And for journal I decide to use Beorg again.
